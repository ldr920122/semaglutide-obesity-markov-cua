# Data Availability Statement

This Markov model manuscript uses publicly available aggregate data and published literature sources. No patient-level hospital data are included in the current package.

The current package includes the executable model code, model input table, base-case results, probabilistic sensitivity analysis outputs, CEAC source data, deterministic sensitivity analysis table, price-threshold scenario table, exact price-threshold summary, effect-retention schedule, retained-effect structural sensitivity table, parameter-source map, and figure source files. These files should be included as supplementary materials or made available according to the target journal policy.

For Frontiers submission, the final Data Availability Statement should insert the actual repository URL after the authors deposit the reproducibility package in OSF, GitHub, Zenodo, Figshare, or an institutional repository. No repository URL is fabricated in this package.
