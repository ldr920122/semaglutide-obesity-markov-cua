# Data Availability Statement

This Markov model manuscript uses publicly available aggregate data and published literature sources. No patient-level hospital data are included in the current package.

The current package includes the executable model code, model input table, base-case results, probabilistic sensitivity analysis outputs, CEAC source data, deterministic sensitivity analysis table, price-threshold scenario table, exact price-threshold summary, effect-retention schedule, retained-effect structural sensitivity table, parameter-source map, and figure source files.

The public reproducibility repository is available at:

https://github.com/ldr920122/semaglutide-obesity-markov-cua

Access date: 2026-06-28.
